## Personal website built by Hugo

## Setup and build steps

```sh
npm run debug   # debug the website
npm run deploy  # deploy the website 
```

## Reference

- [Hugo v0.80.0](https://gohugo.io/) - an extremely fast static site generator
- [Bootstrap 5.0.1](https://getbootstrap.com/docs/5.0/getting-started/introduction/) - responsive front-end library
- [Font Awesome 5.15.2](https://fontawesome.com/) - vector icon library
